# ♟️ Java Chess Game

A fully functional Chess game implemented in Java, focusing on **Object-Oriented Programming (OOP)** principles and complex logic handling.

## 🚀 Key Features
* **Full Move Validation:** Implemented specific movement rules for all pieces (Pawn, Rook, Knight, Bishop, Queen, King).
* **State Detection:** Automatic detection of Check, Checkmate, and Stalemate conditions.
* **OOP Architecture:** Utilized Inheritance, Polymorphism, and Encapsulation to create a modular and maintainable codebase.
* **AI System:** Implemented an AI engine based on the Minimax algorithm with Alpha-Beta pruning to optimize decision-making and search depth.

## 🛠️ Tech Stack
* **Language:** Java
* **Concepts:** Object-Oriented Programming, Logic Design, Data Structures.

## 📂 How to Run
1. Clone the repository.
2. Compile the source files in the `src` folder.
3. Run the Main class to start the game.
